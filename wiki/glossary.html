Glossary

Short, plain-language explanations of the terms used across these games' rules pages.

A way of drawing a partition (a whole number split into a sum of smaller whole numbers, largest first) as rows of stacked boxes: the first row has as many boxes as the largest number in the split, the next row has as many as the next number, and so on, so every row is no longer than the one above it.
A game where both players are allowed to make exactly the same moves at any given point — the only thing that changes turn to turn is whose turn it is. Most games on this site are impartial.
A game where the two players are allowed different moves from each other — for example, one player may only remove rows while the other may only remove columns.
The standard win condition: whoever makes the last possible move wins the game.
The opposite win condition: whoever makes the last possible move loses the game instead. ("Misère" is French for "misery" — it's pronounced "mi-ZAIR".)
The rightmost cell of a row in a Young diagram, but only if no other row directly below it has the exact same length — if several rows in a row share the same length, only the bottom one of that group counts as a corner.
For a chosen cell, all of the cells to its right in the same row plus all of the cells below it in the same column (including the cell itself). Removing a hook can leave the diagram in one piece or split it into two.
